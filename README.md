# 🐱🐶 くぅの食事管理アプリ

このリポジトリは **アプリの公開URLとリダイレクトのみ** を管理しています。
ソースコードは別の Private リポジトリで管理されています。

## アクセス

**新URL**: <https://cat-food-app.pages.dev/>

旧URL（このリポジトリの GitHub Pages）にアクセスすると、自動で新URLにリダイレクトされます。

## 機能

- 猫・犬の食事カロリー計算
- 商品データベース（フード成分表撮影 → AI自動入力）
- 多頭飼い対応（猫・犬それぞれ独立データ）
- バックアップコードでデータ保存・復元
- ライセンスコード会員制

## 配信先

- 本番: Cloudflare Pages (`cat-food-app.pages.dev`)
- リダイレクト用: GitHub Pages (`qoo0902.github.io/cat-food-app`)
